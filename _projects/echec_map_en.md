## Project Description
This project is a streamlit application that maps all the game bars in Paris, France with the detail of the board games they offer. You can also search for a specific game bar or a specific game, and interact with other users, like a personal social media platform for board game lovers !

Disclaimer : it's not perfect, but it's our attempt of creating our first functional app!
